数组

空间名就是数组的名字   你可以根据数组的下标获取数组中的变量

在C语言中，数组是由基本数据类型按照一定规则组成的，称其为构造数据类型
构造数据类型：数组、结构体、共用体等

6.2  一维数组
数组是相同数据类型的有序数据集合
可以是int类型、float类型、doubel类型、char类型

有序数据：数组中的每一个元素在集合中都是按序排列的，排列的位置用下标来表示
带有一个下标的称为一维数组  带有两个下标的称为二维数组   多维数组

6.2.1  一维数组的定义及引用
	一维数组的一般表达式：
		类型名 数组名[常量表达式]	例如：float score[N]   N=50
		int a[6];
		1）定义了一个int类型的数组   数组名是a   有六个int类型的值
		2）定义数组的是方括号  方括号了里面的值是常量表达式
			错误的书写方式：  int a(10); int a{10};
		3)常量表达式  表示数组元素的个数即数组的长度
			C语言规定：数组的下标是从0开始的，下标的最大值：常量表达式-1
				例子：int a[6];   a[0] a[1] a[2] a[3] a[4] a[5]
		4）常量表达式可以是整形的常量或者是符号常量：
				例子  int a[6];      float score[N]   #define N 50 
		5)和普通变量一样，数组定义时同样要分配存储空间（数组分配的空间是连续的）
		6）数组的连续定义  多个数组用逗号隔开
			例子：  int a[5],b[10];
			
	2.一维数组元素的引用
		与简单的变量不同，不能整体的引用一整个数组 ，只能引用数组的某一个
	元素。如何去引用数组中的元素：通过下标去引用相关的元素
		例子：int a[10];     print("%d",a[4]);
		一般表达式：	数组名【下标】
	注意的地方：a[1.6] = 10;  错误   /*下标不能为小数*/

小练习：定义一个数组，然后用循环赋值，单独取出其中一个数据


6.2.2 如何初始化一维数组
	初始化数组的一般表示形式：
		类型名 数组名【数组长度（常量表达式）】={初始表}；
	1）数组元素的初始值依次写在花括号中，数据之间用逗号分开。
		例子：	int a[5]={0,1,2,3,4};
	当初值的个数与数组的长度相等时就可以省略掉数组长度：（柔性数组）
		例子：      int  a[] ={0,2,4,5,7,8,9}；
			会根据初始值的个数来定义数组的长度

	2）静态存储的数组如果没有在定义时初始化，系统会自动给数组所有的元素赋值为0		
	3）动态存储的数组如果没有在定义时初始化，系统会随机给数组的元素赋值
	4）可以只给数组的前几个元素赋值   其余元素自动赋值为0
	5）将数组中的元素全部赋值为0  ------ int a[6] = {0};
	6)  初始化时，初始值的个数不能大于常量表达式
	7）初始表中只能为常量，不能是变量，即使赋值的变量也不行
